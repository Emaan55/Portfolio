My Projects

Amazon Clone Website

This project is a fully functional clone of the Amazon homepage. It features a responsive design with a top navigation bar, search bar, and account options. Below a main banner, various product categories are displayed, including Clothes, Beauty Picks, Pet Care, Furniture, Toys, Electronics, and Fashion Trends. This project showcases proficiency in HTML, CSS, and JavaScript, offering a clean and user-friendly interface.

Cute Innovations Co - Landing Page

This project highlights web design and development skills, emphasizing an engaging and visually appealing user experience.
